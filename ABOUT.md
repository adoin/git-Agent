Git Agent 是面向日常 Git 工作的桌面客户端，提供工作区、提交历史、分支、远程仓库和冲突处理的一体化界面。

## 主要能力

- 管理多个工作空间与仓库
- 暂存、提交、拉取、推送和分支操作
- 浏览提交图谱、文件差异与历史
- 处理合并、变基和三方冲突
- 支持 Git Flow、Git LFS、补丁与仓库基准测试

项目持续开发中。功能行为以当前版本和 Git 实际执行结果为准。

<!-- lang:en-US -->
Git Agent is a desktop client for everyday Git work. It combines workspaces, commit history, branches, remotes, and conflict resolution in one interface.

## Key capabilities

- Manage multiple workspaces and repositories
- Stage, commit, pull, push, and manage branches
- Browse commit graphs, file diffs, and history
- Resolve merges, rebases, and three-way conflicts
- Use Git Flow, Git LFS, patches, and repository benchmarks

The project is under active development. Behavior follows the current release and the result returned by Git.
