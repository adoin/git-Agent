## 已完成：Edit 动作

目标：计划中可选择“编辑”。Git 在该提交处暂停，用户修改工作区、暂存后继续变基。

验收：

1. 选择某条普通提交为“编辑”，执行计划。
2. 工作区进入变基中间态，提交区显示继续、跳过、中止。
3. 用户修改文件并暂存后点击继续，历史生成新的提交。
4. 合并提交不允许选择“编辑”。
